# MCP Server

The MCP server (`packages/mcp-server`) exposes schema validation tools for use in any MCP-compatible client (Cursor, Claude Desktop, Claude Code, etc.).

## Tools

| Tool | Description |
|------|-------------|
| `validate_schema` | Validate a JSON schema against a provider's rules |
| `fix_schema` | Auto-fix common schema issues for a provider |
| `list_groups` | List all supported provider groups and their models |
